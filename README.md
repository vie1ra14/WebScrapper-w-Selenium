# Web Scraper com Selenium, SQLite e Streamlit

Este projeto é um **web scraper automatizado** desenvolvido em Python, utilizando **Selenium** para coleta de dados, **SQLite3** para armazenamento local e **Streamlit** para visualização dos dados em uma interface web simples e intuitiva.

A aplicação foi pensada para extrair informações de sites que utilizam JavaScript, salvar os dados de forma organizada e permitir a análise através de uma interface amigável.



## Tecnologias Utilizadas

- **Python 3.11+**
- **Selenium** 
- **SQLite3** 
- **Streamlit** 
- **Pandas**
- **WebDriver (ChromeDriver)**



## Instalação e Configuração

### 1. Clonar o repositório
```bash
git clone https://github.com/seu-usuario/nome-do-projeto.git
cd nome-do-projeto
```
### 2. Instalar dependências
```bash
pip install -r requirements.txt
```
### 3. Instalar ultima versão do Chrome

## Como executar
### 1. Executar o Web Scraper
```bash
python -m scraper.scraper
```

### 2 . Executar a Aplicação Streamlit
```bash
streamlit run dashboard/app.py (http://localhost:8501)
```
